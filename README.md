# cybersecurity-home-lab
Hands on cybersecurity home lab focused on networking and security fundamentals.

This repository documents my hands-on cybersecurity home lab projects.
The goal is to gain practical experience in networking, system security,
and defensive/offensive techniques.

## Lab Environment
- VirtualBox
- Windows 10 (Target System)
- Kali Linux (Linux-based Attacker & Analysis System)

## Skills Practiced
- Virtualization
- Network isolation
- Firewall configuration
- User management
- Vulnerability scanning
- Log analysis

## Labs
### Lab 1: Basic Network Security Lab
- Created isolated internal network
- Configured Windows firewall
- Ran reconnaissance from Kali Linux
- Observed system logs on Ubuntu
